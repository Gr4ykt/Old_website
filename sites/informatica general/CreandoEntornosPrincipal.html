Conceptos


¿Qué es un sistema operativo?

Bueno, para empezar, un sistema operativo es un conjunto de programas el cual permite manejar diversas partes del hardware para la interacción con el usuario, para poder hacer esta comunicación están los drivers o controladores, en pocas palabras el sistema operativo es la imagen de la computadora la cual permite la interacción con el usuario y la máquina. Bien si hay diversos sistemas operativos como MAC, Windows, FreeBSD, MS-DOS, etc. Quiero destacar Linux.


¿Qué es GNU/Linux?

GNU/Linux es un sistema operativo o más bien, un kernel de código abierto desarrollado por Linus Torvalds basándose en los sistemas Unix, este kernel al ser un software libre(No significa lo mismo que código abierto), este puede ser utilizado para el desarrollo de diversos sistemas operativos(tal es el caso de Debian, la cual es una distribución de Linux); ya que entra este concepto, así se les denomina a estos sistemas desarrollados bajo un kernel Linux, distribuciones, existen diversas versiones y distribuciones, cada una con algo en especial, desde diversos gestores de paquetes, entornos caseros o más profesionales, ejemplo de estos son muchos, casos como Arch Linux(Este es complicado para los novatos, pero si no temes a los retos adelante). Kali Linux o parrot, distribuciones dedicadas al hacking y pentensting, o por último, Ubuntu, una distribución amigable muy cómoda para la computadora convencional(incluso yo lo utilicé como sistema operativo principal mucho tiempo).

Pues así es, para GNU/Linux gustos y colores, hasta la interfaz gráfica que esta tenga, o terminal, pueden ser completamente personalizables a tu gusto y para el uso que quieras, si hasta los mismos comandos puedes programarlos tu mismo, aparte de tener una inmensa comunidad dispuesta a ayudar.


¿Qué es una máquina virtual?

Una máquina virtual, o virtualización, es la "emulación" de un sistema operativo sobre nuestro sistema base, ósea, el sistema operativo es ejecutado a través de un programa de virtualización(VirtualBox, VMWare, etc) para que este se ejecute como un programa(wow, una computadora dentro de una computadora, mal chiste....).


¿Qué es VirtualBox?

VirtualBox es un programa de virtualización, es el que yo personalmente utilizo diariamente, aunque muchos prefieren VMWare, pero por comodidad y adaptación, prefiero VirtualBox, fue el primer programa de virtualización que utilice y con el que mejor me manejo, conozco sus mañas y se me hace cómodo, aunque tiene varios conceptos que te invito a que averigües tu mismo, ya que son muchos, como los discos virtuales, la exportación e importación de cajas, etc.


¿Qué es Debian?

Debian es una distribución GNU/Linux, sistemas operativos antes mencionado como Kali, Parrot y Ubuntu, son sistemas basados en Debian, estos se caracterizan especialmente por su gestor de paquetes aptitude o apt(Este será un tema que se tocara más adelante, pero si quieres investigarlo, adelante), Elegí esta distribución porque aparte de ser sencilla de instalar y utilizar, fue la primera distro que utilice, por ello, estoy más familiarizado.
